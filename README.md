# Mixing App

This Mixing App allows users to create and manage tracks on a mixing deck. Users can add new tracks, remove existing tracks, and visualize their mixing setup.

## Getting Started

To run the Mixing App locally, follow these steps:

1. Clone the repository:
   
Install dependencies:

npm install
Generated


Start the development server:

npm run dev
Generated


Open your browser and navigate to http://localhost:3000 to view the Mixing App.

Features
Mixing Deck: The main interface where users can view and manage their tracks.
Add Track: Users can add new tracks to the mixing deck by entering the track name.
Remove Track: Users can remove tracks from the mixing deck by clicking the "Remove" button.
